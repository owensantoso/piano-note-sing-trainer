# Roadmap

Use this for sequence and rationale, not as a dumping ground.

## Now
- Build the smallest mobile web app baseline.
- Prototype microphone capture and monophonic pitch detection.
- Validate the sing-first, play-one-piano-note loop in a quiet room.

## Next
- Build the full-screen pitch arena UI with fake data, then connect real pitch input.
- Add stable note capture, confidence thresholds, and semitone distance feedback.
- Manually test iPhone Safari and Android Chrome.

## Later
- Optional Web MIDI input for browsers and devices that support it.
- Cents-level tuning feedback after semitone matching is reliable.
- Sampled piano tones or reference playback.
- Local practice history and trends.
- Additional exercises such as app-plays-target, intervals, scales, and solfege modes.
